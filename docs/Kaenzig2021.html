1 Overview

This document replicates the main empirical results of (känzig2021?) , which identifies oil supply news shocks using a high-frequency external instrument (proxy SVAR). The instrument captures exogenous variation in oil supply driven by OPEC production decisions, measured as oil futures price changes in a narrow window around OPEC announcements.

8 IV Identification (Manual Two-Stage)

u   <- residuals[(nrow(residuals) - length(iv_oil) + 1):nrow(residuals), ]
T1  <- nrow(u)
sigma <- (t(u) %*% u) / (T1 - 1 - p - N * p)
S     <- t(chol(sigma))

# Stage 1: regress u_p on instrument
ols1 <- fOLS(y = u_p_fin, X = iv_oil, c = 0)
uhat <- ols1$fitted_partial

# Stage 2: regress other residuals on uhat to get s_q / s_p
u_q_fin <- u[(nrow(u) - nrow(iv_oil) + 1):nrow(u), -1]
sq_sp <- solve(t(uhat) %*% uhat) %*% t(uhat) %*% u_q_fin

# Structural impact vector (normalised to unit oil price response)
s <- c(1, sq_sp[1], sq_sp[2], sq_sp[3], sq_sp[4], sq_sp[5])

# Structural IRFs
ivirf <- matrix(0, nrow = N, ncol = hor + 1)
for (h in 1:(hor + 1)) {
    ivirf[, h] <- wold[, , h] %*% s
}

13 Proxy IV Restricted to OPEC Months

As a robustness check, the structural impact vector is re-estimated using the proxy IV approach but restricted to OPEC announcement months only (where the instrument is non-zero). Specifically, both the proxy and VAR residuals are centred within the OPEC subsample, and \(b_1\) is estimated as \(\hat{b}_1 = (z'z)^{-1} z'U\) where \(z\) and \(U\) are the centred proxy and residuals in announcement months. Confidence bands use the same MBB bootstrap as the main proxy SVAR, resampling residuals and proxy jointly.

# Treatment indicator: 1 = OPEC announcement month, 0 = other
indsR1 <- as.integer(iv_oil[, 1] != 0)
cat("OPEC announcement months:", sum(indsR1), "of", length(indsR1), "\n")
#> OPEC announcement months: 117 of 417

15 Internal Instruments: Cholesky on Augmented VAR

An alternative to the proxy SVAR is to augment the VAR with the instrument as the first variable and apply a recursive (Cholesky) ordering. Under the assumption that the instrument responds only to oil supply news contemporaneously — and is not driven by the macro variables within the same period — the first Cholesky shock recovers the oil supply news shock. Pre-sample periods where the instrument is missing are set to zero (same truncation as iv_oil_trunc).

N_aug        <- N + 1
y_aug        <- cbind(iv_oil_trunc, finaldata)
varnames_aug <- c("Oil Instrument", varnames)

var_aug  <- fVAR(y_aug, p, c)
S_aug    <- t(chol(var_aug$sigma))
wold_aug <- fwoldIRF(var_aug, horizon = hor)

point_irf_aug <- fcholeskyIRF(wold_aug, S_aug)
tic()
boot_aug <- fbootstrapChol(
    y          = y_aug,
    var_result = var_aug,
    nboot      = 1000,
    horizon    = hor,
    prc        = 90,
    prc2       = 68,
    n_threads  = 3
)
toc()
#> 1.529 sec elapsed

16 Weak-IV Robust Inference (Montiel-Olea, Stock, Watson 2021)

The MBB bootstrap assumes a strong instrument. Montiel-Olea, Stock and Watson (2021) provide confidence sets that remain valid regardless of instrument strength. Two types are reported:

  • Delta method (inner, darker band): plug-in inference using the asymptotic variance of the IV estimator.
  • Anderson-Rubin (outer, lighter band): weak-IV robust confidence set; bounds solve a quadratic inequality and may be unbounded if the instrument is very weak.
msw <- fMSW(
    var_result = var_result,
    Z          = iv_oil,
    finaldata  = finaldata,
    adjustu    = adjustu,
    hor        = hor,
    nvar       = 1,
    scale      = 1,
    confidence = 0.9,
    NWlags     = 0
)

cat("Wald statistic (HAC-robust F):", round(msw$Waldstat, 2), "\n")
#> Wald statistic (HAC-robust F): 11.67
fplotirfMSW(
    msw_result = msw,
    varnames   = varnames,
    shockname  = shockname,
    scale      = 10,
    facet_ncol = 3,
    line_color = 'black',
    ribbon_fill_ar = tidyMacro_colors[1],
    ribbon_fill_dm = tidyMacro_colors[3],
    ribbon_alpha_ar = 0.45,
    ribbon_alpha_dm = 0.45
) +
  labs(x = NULL, y = NULL)
